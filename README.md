## index.html이 하는 일

스크롤 한 장에 아래 순서로 들어 있습니다. 모든 입력은 **브라우저 localStorage에 자동 저장**되어
새로고침해도 남습니다. (서버·로그인 없음, 기기/브라우저마다 따로 저장됨)

1. **감성 표지** — 둥근 시트 카드 + 손잡이 바, 포근나루 팔레트.
2. **월간 달력** — 날짜 아래 작은 점으로 일정 표시(활동 색), 오늘은 테라코타 원.
   `‹ ›` 로 달 이동. 날짜를 누르면 아래 "일별 패널"이 그날로 바뀜.
3. **이번 달 독서**(중심) — 영역별(도란도란·나루의 밤·개인 도서·디퍼 등) 색 점 + 이름 + **진행 원**.
   원을 누르면 진행도가 채워짐. 독서 항목 추가/삭제 가능, 칸 수도 항목마다 지정.
4. **일별 패널** — 선택한 날의 일정 보기/추가/삭제, 습관 체크, 한 줄 메모.
5. **이번 달 습관** — 습관 5종(선릿모닝·스페인어 원서·고전 읽기·원고·9시 이전 마무리)을
   한 달치 칸으로. 칸을 누르면 채워지고 "○일" 합계 표시. 미래 날짜는 흐리게.

### 디자인
- 팔레트: 크림 `#F1E7DA` / 카드 `#FCF6ED` / 테라코타 `#C8845A` / 다크브라운 `#3C2415`.
- 폰트: 제목 **Gowun Batang**(명조), 본문 **Apple SD Gothic / Noto Sans KR**(고딕).
- 활동 색: 미사·신앙(주황) / 전례·봉사(하늘) / 독서모임(분홍) / 개인독서(보라) / 클래스·나루(노랑) / 디퍼·작업(네이비).

### 데이터(localStorage 키)
- `gabi-reads` — 독서 항목  ·  `gabi-habits` — 습관 체크  ·  `gabi-events` — 일정  ·  `gabi-memos` — 메모
- 7월 알려진 일정(선릿모닝 1기 개강 7/1, 나루의 밤 마감 7/2, 매주 목 디퍼 줌, 금 미사·성경공부, 토 자모회 등)은
  코드에 **시드(기본값)**로 내장. 나머지는 햇살이 날짜를 눌러 직접 추가.
