## index.html이 하는 일

스크롤 한 장. 모든 입력은 **브라우저 localStorage에 자동 저장**(서버·로그인 없음, 기기/브라우저마다 따로).

1. **감성 표지** — 둥근 시트 카드 + 손잡이 바, 포근나루 팔레트.
2. **월간 달력** — 날짜 아래 작은 점으로 일정 표시(활동 색), 오늘은 테라코타 원. `‹ ›`로 달 이동.
   날짜를 누르면 아래 "일별 패널"이 그날로 바뀜.
3. **이번 달 독서**(중심) — **도서명**으로 표시 + 진행 원(누르면 채워짐). 추가/삭제, 칸 수 지정 가능.
4. **일별 패널** — 그날 일정 보기/추가/삭제, 습관 체크, 한 줄 메모.
5. **이번 달 습관** — 습관 3종을 한 달치 칸으로. 칸을 누르면 채워지고 "○일" 합계 표시. 미래 날짜는 흐리게.

### 독서·습관 구성 (2026년 7월 기준)
- 독서(도서명 — 토론별 색):
  - 절창(구병모) → 도란도란 **핑크**
  - 마음(나쓰메 소세키) → 나루의 밤 **바나나**
  - 에디톨로지(김정운) → 디퍼 **네이비**
  - 파리의 노트르담(위고) → 개인 **라일락**
  - 프린키피아(뉴턴) → **바나나**. 그림 하나로 2시간 토론하기도 해 **토론 일정(7/12·26)만 남기고 독서 트래커엔 미표시**
- **독서 진행 칸 = 그 책의 7월 독서 일정 일수**(에디 5·마음 3·절창 4·파리 9). 칸에 마우스 올리면 날짜·분량이 뜨고, 그날 분량을 읽으면 그 칸을 체크.
- 습관 3종: 선릿모닝(레몬) · 스페인어(다크올리브) · 독서(브라운).

### 일정 색(카테고리)
도란도란(핑크) · 나루의 밤(바나나) · 디퍼(네이비) · 개인(라일락) · 성당·신앙(주황) · 병원(세이지) · 아이(하늘).

### 디자인
- 팔레트: 크림 `#F1E7DA` / 카드 `#FCF6ED` / 테라코타 `#C8845A` / 다크브라운 `#3C2415`.
- 폰트: 제목 **Gowun Batang**(명조), 본문 **Apple SD Gothic / Noto Sans KR**(고딕).

### 데이터(localStorage 키)
- `gabi-reads` 독서 · `gabi-habits` 습관 체크 · `gabi-events` 일정 · `gabi-memos` 메모 · `gabi-seedver` 시드 버전.
- **일정·독서는 코드에 시드(기본값)로 내장.** 매달 갱신할 때는 `index.html`의 `seedEvents()`·`seedReads()`를
  새로 채우고 **`SEED_VER` 값을 올리면** 다음 접속 때 새 시드가 적용됨(습관 체크·메모는 보존).
- 7월 일정은 가비 알림장(`클로드 에이전트/가비 알림장/2026-7월.md`)을 반영해 입력함.
  - 디퍼 2기 문법: 매주 월 21:00(7/6·13·20·27) · 1기 수사: 목 21:00(7/2·9·16, 에디톨로지 토론) + 7/25 수료
  - 도란도란 09:30(7/2 파리의 노트르담·7/23 절창) · 나루의 밤 20:00(7/21 마음) · 프린키피아 20:00(7/12·26)
  - 성당·병원·아이 일정 다수
  - **독서 분량 매일 배분**(마감 하루 전 완료 역산): 에디톨로지(7/1~13)·마음(7/17~20)·절창(349p, 7/14·15·16·22 분할)·파리의 노트르담(1권 25장, 일·화·목 오전 페이스)
